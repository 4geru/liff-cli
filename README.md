# liff-cli

LIFF(LINE Front-end Framework) cli tool  
https://developers.line.me/en/docs/liff/overview/

# Installation
```
$ npm install -g liff
$ export LINE_ACCESS_TOKEN={YOUR_LINE_ACCESS_TOKEN}
```

# Usage

### list
```
$ liff list
```
